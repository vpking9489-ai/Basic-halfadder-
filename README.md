# Half Adder in Verilog

## Author
Vraj Patel

## Description
This project implements a Half Adder using Verilog HDL.

## Inputs
- a
- b

## Outputs
- sum = a XOR b
- carr = a AND b

## Files
- halfadder.v → Design module
- tb_halfadder.v → Testbench for simulation

## Simulation
All input combinations are tested using testbench.

## Tools Used
- ModelSim
- Quartus
